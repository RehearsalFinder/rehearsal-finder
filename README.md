# Rehearsal Finder

Rehearsal Finder (tentative title) is a web application that connects musicians with inspirational environments in which to create.

## Necessary Dependencies
**ember install**
**ember-cli-eslint** 
**ember-simple-form** 
**ember-cli-template-lint** 
**ember-cli-sass** 
**ember-cli-sass-lint** 
**ember-cli-flash**
